# Equipo 07 — Clasificación de enfermedades en hojas usando PlantVillage

Este repositorio contiene la implementación del proyecto de Deep Learning del Equipo 07. El objetivo es realizar fine-tuning y comparación de modelos de visión por computadora para clasificar enfermedades en hojas de plantas del dataset PlantVillage.

## Estructura del repositorio

```text
Equipo_07_Clasificacion_PlantVillage/
├── README.md
├── requirements.txt
├── .gitignore
├── notebooks/
│   ├── 01_entrega_02_version_inicial.ipynb
│   └── 02_entrega_03_version_final.ipynb
├── app_huggingface/
│   ├── v1/
│   └── v2_final/
└── referencias/
    └── fuentes.md
```

## Notebooks

- `notebooks/01_entrega_02_version_inicial.ipynb`: primera versión del flujo de implementación del proyecto.
- `notebooks/02_entrega_03_version_final.ipynb`: versión final del entrenamiento, evaluación y pruebas del modelo.

## Aplicación en Hugging Face

- `app_huggingface/v1/`: primera versión de la aplicación demostrativa en Gradio.
- `app_huggingface/v2_final/`: carpeta reservada para la versión final de la aplicación.

## Ejecución recomendada

1. Crear un entorno de Python o usar Google Colab.
2. Instalar las dependencias principales:

```bash
pip install -r requirements.txt
```

3. Abrir y ejecutar el notebook final:

```text
notebooks/02_entrega_03_version_final.ipynb
```

4. Verificar que el entrenamiento, la evaluación y las métricas se ejecuten correctamente.
